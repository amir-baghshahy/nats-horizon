nats-horizon is built on the shoulders of giants:

- **[NATS](https://nats.io/)** — The high-performance messaging system

- **[Gin](https://gin-gonic.com/)** — HTTP web framework
- **[React](https://react.dev/)** — UI library
- **[Vite](https://vitejs.dev/)** — Build tool
- **[TanStack Query](https://tanstack.com/query)** — Data fetching
- **[TailwindCSS](https://tailwindcss.com/)** — Styling

- [nats-nui](https://github.com/nats-nui/nui) — Performance benchmarks
- [nats-console](https://github.com/KLogicHQ/nats-console) — Feature completeness
